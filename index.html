Capstone Project
Foods Of Canada

Foods of Canada is an online interactive geographical data base to be used as a tool to look up and learn about the foods of Canada. The user interface presents a map of Canada that functions as a search engine. The user could click on a point on the map/identified location (zoom in and out), and products that belong in that location are showed on a side bar.
If the user selects a product, relevant information about that product is displayed on the side bar. Also, there is a search with filters that users can select to narrow down their searches by region, season, or by-product. Other functionalities included are: Users can create an account, save favorite products, and have their own pantry where they can save a list of products according to their preferences. The back-end was implented using Spring Framework and the Front-end on React. This project was developed as a par of a team and I was focused on back-end development.
